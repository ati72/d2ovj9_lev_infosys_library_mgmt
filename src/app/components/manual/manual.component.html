How to use this application

Handling Members

Listing members

Click the members menu on the sidenav panel. The members contained in the database should be listed. You can search for members using the Search input field or sort them by clicking the table column headers. The table is paginated, you can choose how much items per page you want to display.

Adding a new member

To add a new member navigate to the members component by clicking the members menu on the sidenav panel. Click the Add New Member button and fill in the form in the dialog panel, then click the Save button. The input fields are validated.

Deleting and updating members

Click the trash icon to delete a member. You can only delete members who don't have any rented items! If they have rented items, the trash icon won't be displayed. To Update a member click on the update icon, fill out the form and click Update.

Handling items

Listing items

Click the Books menu on the sidenav panel. The items contained in the database should be listed. You can search for items using the Search input field or sort them by clicking the table column headers. The table is paginated, you can choose how much items per page you want to display.

Adding a new item

To add a new item navigate to the items component by clicking the Books menu on the sidenav panel. Click the Add New Item button and fill in the form in the dialog panel, then click the Save button. The input fields are validated.

Deleting and updating items

Click the trash icon to delete an item. You can only delete items which aren't currently rented! If they are rented, the trash icon won't be displayed. To Update an item click on the update icon, fill out the form and click Update.

Renting and returning items

Renting

Navigate to the Books menu on the sidenav panel. If an item's status is "Available" it can be rented. Click on the shopping cart icon to assign it to a member. Type in the ID number of the member and click Rent. If the member already has 6 items rented, he won't be able to rent another one.

Returning

Navigate to the Book menu on the sidenav panel. If an item's status is "Rented" it can be returned. Click the icon to return it.
